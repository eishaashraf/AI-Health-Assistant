# 🩺 AI Health Assistant

An AI-powered health support application built with Streamlit that provides users with basic health-related tools including BMI calculation, symptom guidance, medical imaging information, healthy lifestyle recommendations, and first-aid assistance.

## 🚀 Project Overview

Healthcare information can be difficult to understand for many people. AI Health Assistant aims to provide a simple and interactive platform where users can access basic health guidance and educational resources in one place.

## ✨ Features

### 1. BMI Calculator
- Calculates Body Mass Index (BMI)
- Provides basic weight category information

### 2. AI Symptom Checker
- Allows users to enter symptoms
- Provides possible health guidance using AI assistance

### 3. Medical Imaging Assistant
- Explains different medical imaging techniques:
  - MRI
  - CT Scan
  - PET Scan
  - Ultrasound

### 4. Healthy Lifestyle Guide
- Provides suggestions related to:
  - Nutrition
  - Exercise
  - Healthy habits

### 5. First Aid Assistant
- Provides basic first-aid information for common situations

## 🛠️ Technologies Used

- Python
- Streamlit
- OpenAI API / OpenRouter API
- Python-dotenv

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/eishaashraf/AI-Health-Assistant.git
```

Move into the project folder:

```bash
cd AI-Health-Assistant
```

Install the required packages:

```bash
pip install -r requirements.txt
```
## ▶️ How to Run

Create a `.env` file and add your OpenRouter/OpenAI API key.

Example:

```text
OPENROUTER_API_KEY=your_api_key_here
```

Run the application:

```bash
python -m streamlit run app.py
```
